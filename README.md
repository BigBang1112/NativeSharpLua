# NativeSharpLua

[![Nuget](https://img.shields.io/nuget/v/NativeSharpLua?style=for-the-badge&logo=nuget)](https://www.nuget.org/packages/NativeSharpLua/)
[![GitHub release (latest by date including pre-releases)](https://img.shields.io/github/v/release/BigBang1112/NativeSharpLua?include_prereleases&style=for-the-badge&logo=github)](https://github.com/BigBang1112/NativeSharpLua/releases)

A simple Lua wrapper with NativeAOT support and transparent C lib build.

Current Lua version: **5.5.0**

Supported runtimes:
- `win-x64`
- `win-x86`
- `linux-x64`
- `linux-arm`
- `linux-arm64`
- `browser-wasm`
